# 你好，世界！

欢迎来到我的博客。

## 为什么写博客？

在这个信息爆炸的时代，**写作是最好的思考方式**。写博客不是为了让别人看到，而是为了整理自己的思绪，记录成长的过程。

## 这个博客的特点

- ✨ **极简设计** — 专注阅读，没有多余的干扰
- 🌓 **暗色模式** — 自动跟随系统，也可以手动切换
- 📝 **Markdown** — 用熟悉的语法写作
- ⚡ **纯静态** — 托管在 GitHub Pages，快速加载

## 代码示例

```python
def greet(name: str) -> str:
    return f"你好，{name}！"

print(greet("世界"))
```
